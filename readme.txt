== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce > Settings > Payments Tab, and toggle “ON” the Yesh Invoice - Gateway payment method that the customer will see during checkout.

You can clear credit cards with Tranzila, CardCom, PeleCard, Meshulam, PayPal, YaadPay, Max Bussiness. All you have to do is connect to the plugin in Yesh Invoice Market place.
